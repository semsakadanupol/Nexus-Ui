Modern Navbar

The Nexus-UI navbar is designed with a professional, modern aesthetic that instantly elevates your project. Built with smooth animations, responsive design, and theme support.

✨ Modern Design

Subtle gradients and glassmorphism effects create a professional look that works across all themes.

⚡ Smooth Animations

Entrance animations, hover effects, and staggered delays create engaging interactions.

🎨 Theme Ready

Fully supports light, dark, and high-contrast themes with smooth CSS transitions.

> Key Features

>
  • Sticky positioning with slide-down animation on page load
  • Gradient brand with interactive hover effects and scale transforms
  • Animated underline on nav links with scaleX animation
  • Responsive mobile menu with smooth hamburger animation
  • Dropdown menus with smooth transitions and scale animations
  • Full theme support with CSS variables for easy customization
